## Getting Started

### Prerequisites

```bash
# Example:
Node.js
npm install -g nodemon
```

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Germa-Studio/sikaetan-Be.git
cd sikaetan-Be
```

2. Install dependencies:

```bash
npm install
```

3. Start the application:

```bash
npm start
```
OR
```bash
nodemon
```
